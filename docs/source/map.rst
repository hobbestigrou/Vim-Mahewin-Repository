Map
===
Few pratical map.

Insert date
-----------
Insert the current date in insert map:

.. code-block:: vim

    <leader>date

Use french format like that 08/02/2014.

Git map
-------
Few git map.

Pull a code
___________
To pull a git repository in the normal mode:

.. code-block:: vim

    <leader>gp

Use the current branch.

Push code
_________
To push code in the normal mode:

.. code-block:: vim

    <leader>gpu

Use the current branch.
